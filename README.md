Projet POO - Rocher - Vigneux - Vrignon

De quoi avons-nous besoin ?

Ecran des personnages : un button jouer :

-> Créer une nouvelle salle.
	-> title
	-> lieu
	-> portes = nb; -> permet d'acceder à une nouvelle salle
	-> coffres = nb; -> gains
	-> monstres = nb; -> combat

-> Créer un nouveau personnage de classe Personnage.
  -> - atk
     - mag
     - def
     - hp
     - wp

     Méthodes de chaques personnages :
	-> Observer
	-> Passer une porte
	-> Ouvrir un coffre
	-> Attaquer
	-> Fuir

-> Choisir une classe de personnage :

  -> ajoute des méthodes
	- attaquer (guerrier)
	- lancerSort (magicien)
	- Choix (paladin)

Méthodes actions :

- Observer : wip
- Passer une porte : change de niveau + créer une salle
- Ouvrir un coffre : gagner un objet aléatoire (armes, potion)
- Attaquer : inflige des dégats
- Fuir : change de niveau ou annule un combat
